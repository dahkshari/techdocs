## **What’s new in Core Product 1.5?**

*\[Feature descriptions state the update and its benefits.\]*

This release includes the following new product features:

### First Feature Title

First Feature Title enables Core Product 1.5 to comply with government security requirements. This ensures Company passes legal audits and avoids fines and penalties.

### Second Feature Title

Second Feature Title enables users to edit jobs while they are running. This reduces time spent waiting for a job to complete before users can make edits.

## **Defects Fixed**

*\[Defect descriptions state the bug and the fix.\]*
 
This release fixes or addresses the following defects:

### First Defect

A defect on the Job Creation page prevented users from selecting a Product to add to the job run. Users can now see all options in the Product dropdown field.

### Second Defect

Job runs were timing out after one minute, which was not long enough to complete a job run. Core Product 1.5 extends the time-out clock to ten minutes.
